# Mobile App Vision

## Objective
Provide rapid donor response and profile management from mobile devices using the same backend API.

## React Native Structure
- `mobile/src/navigation/AppNavigator.tsx`
- `mobile/src/screens/DonorLoginScreen.tsx`
- `mobile/src/screens/DonorRegisterScreen.tsx`
- `mobile/src/screens/DashboardScreen.tsx`
- `mobile/src/screens/EmergencyAlertsScreen.tsx`
- `mobile/src/screens/AppointmentsScreen.tsx`
- `mobile/src/screens/DonationHistoryScreen.tsx`
- `mobile/src/screens/ProfileScreen.tsx`
- `mobile/src/services/api.ts`

## Planned Mobile Flows
1. Donor authentication and secure token storage.
2. Real-time emergency alert feed and one-tap response.
3. Appointment calendar and reminders.
4. Donation history timeline.
5. Availability toggle and notification settings.
